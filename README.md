# The Slice Cake Shop

Buttercream Cakes • Brownies • Cupcakes • Muffins. Freshly baked with love & perfection.

## About
The Slice Cake Shop creates handcrafted baked goods made to order. We specialize in custom buttercream cakes, traybakes, cupcakes, and seasonal treats.

## Our Menu (examples)
- Cakes: Custom celebration cakes (sizes: 6", 8", 10") — flavors: Vanilla, Chocolate, Red Velvet
- Cupcakes: Vanilla, Chocolate, Red Velvet, Salted Caramel
- Brownies & Bars: Fudge brownies, blondies
- Muffins: Blueberry, Banana Nut
